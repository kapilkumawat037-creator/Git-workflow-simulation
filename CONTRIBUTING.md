# CONTRIBUTING GUIDE

## Steps Performed

1. Created Git repository
2. Created main branch
3. Created feature-login branch
4. Modified app.py in feature-login
5. Committed login feature
6. Created feature-dashboard branch
7. Modified app.py differently
8. Committed dashboard feature
9. Merged feature-login into main
10. Merge conflict occurred while merging feature-dashboard
11. Resolved conflict manually
12. Final merge completed successfully

## Git Commands Used

```bash
git init
git branch -M main
git checkout -b feature-login
git checkout -b feature-dashboard
git merge feature-login
git merge feature-dashboard
